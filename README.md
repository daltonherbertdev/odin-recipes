# odin-recipes

This repository is for my recipe book. Skills showcased will include anchors, images, and the proper use of lists and local navigation. 

A few things I learned is how important it is to keep things organized in your code. You want to have things properly nested for yourself or other devs. I also got good practice navigating saved files using the terminal, moving them, renaming them etc.

This project also helped me really understand the flow of Git. Looking at the status, seeing the files I may have forgotten to commit or add to the staging satus. Overall great project and really improved my understanding of the general work flow.
